# Contributing to mrcorpt-web

Thank you for your interest in contributing! We welcome contributions from everyone. Here's how you can help:

## Getting Started

1. **Fork the repository** and clone it locally
   ```bash
   git clone https://github.com/YOUR-USERNAME/mrcorpt-web.git
   cd mrcorpt-web
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Create a new branch** for your changes
   ```bash
   git checkout -b feature/your-feature-name
   ```

## Development Workflow

- Make your changes and test thoroughly
- Keep commits atomic and descriptive
- Follow the existing code style and conventions
- Update documentation as needed

## Commit Messages

Use clear, descriptive commit messages:
- Use present tense ("Add feature" not "Added feature")
- Be specific about what changed
- Reference issues when applicable (e.g., "Fixes #123")

Example:
```
Fix responsive design issue on mobile devices

Resolves #456
```

## Pull Request Process

1. **Push your branch** to your fork
2. **Create a Pull Request** with a clear title and description
3. **Link related issues** in the PR description
4. **Ensure all tests pass** and CI checks are green
5. **Request review** from maintainers
6. **Address feedback** from code reviewers

### PR Guidelines

- Keep PRs focused on a single feature or fix
- Include a clear description of changes
- Add screenshots if UI changes are involved
- Update tests and documentation

## Code Style

- Follow the existing code style in the repository
- Use meaningful variable and function names
- Add comments for complex logic
- Format code consistently

## Reporting Bugs

Found a bug? Please open an issue with:
- Clear description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Environment details (OS, browser, etc.)
- Screenshots if applicable

## Feature Requests

Have an idea? We'd love to hear it! Open an issue with:
- Clear description of the feature
- Use cases and benefits
- Examples or mockups if applicable

## Code of Conduct

This project adheres to a [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.
