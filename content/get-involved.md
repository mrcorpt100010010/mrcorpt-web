Whether you are a programming wizard or just a fan, there's something you can do to help make MrCorpt even better.

## MrCorpt-test

Help MrCorpt improve by installing the MrCorpt-test worker; your computer will run games to test future versions of the engine!

- Windows: [Install the worker GUI **(beta)**](https://github.com/mrcorpt100010010/mrcorpt-test-worker-gui/releases/latest/download/fishtest-worker-gui.exe)
  - The worker GUI is unsigned; if you see "Windows protected your PC", click More info → Run anyway.
- Linux & Cloud: [Server and Worker Docker Image](https://github.com/mrcorpt100010010/docker-mrcorpt-test)
- [Further options and detailed Worker Documentation](https://github.com/mrcorpt100010010/mrcorpt-test/wiki/Running-the-worker)
- [Dashboard](http://example.com/tests)

## Write Code

Do you have a great idea that could improve MrCorpt's playing
strength? Fork the project on GitHub and commit away. Then use MrCorpt-test
to test your change.

- [MrCorpt on GitHub](https://github.com/mrcorpt100010010-mrcorpt/MrCorpt)
